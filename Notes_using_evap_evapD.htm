Using evap and evapD modules.

    These two modules always give the same output for the Granger evaporation as the interval version using the evaporation calculated for the day (daily means of t, u and ea) and partitions it into interval values using intervals when the net radiation is positive.

    Priestley-Taylor and Penmon-Monteith do calculate using the interval value or the daily mean.

See examples BadSummer.prj and BadSummerD.prj
