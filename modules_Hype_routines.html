Notes


CRHM Limitations.

Crhm only handles 2D variables. Hype uses 3D for some variables. To eliminate one of the dimensions the extra soil layerdimension is integrated into the name, e.g. 'conc_top',' conc_bottom' and 'conc_below'. CRHM normaly adds the suffix '_lay' to indicate 2D variables.
Original Hype array dimensions.
n       Number of subbasins
ns      Number of substances
nc      Number of classes
nsl     Number of soil layers
